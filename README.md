En el proyecto demuestra el uso de los métodos especiales __init__ (constructor) y __del__ (destructor) en Python, aplicando una arquitectura orientada a objetos con separación de responsabilidades.

El objetivo

Implementar clases que utilicen constructores para inicializar recursos (como archivos).
Utilizar destructores para liberar o cerrar dichos recursos de forma controlada.
Aplicar una estructura de proyecto limpia basada en capas: modelos, servicios y punto de entrada.
